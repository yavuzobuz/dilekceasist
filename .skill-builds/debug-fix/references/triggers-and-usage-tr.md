# Turkish Triggers And Usage

Use this file to recognize Turkish requests and keep explanations friendly for less experienced users.

## Common Turkish Triggers

- "hata ayikla"
- "hatayi bul"
- "neden patliyor"
- "neden bozuldu"
- "bunu duzelt"
- "fix et"
- "test failini coz"
- "bu sayfa calismiyor"
- "loglara bak"
- "root cause bul"

## Common English Triggers

- "debug this"
- "find the bug"
- "why is this failing"
- "fix the regression"
- "investigate this error"
- "trace the root cause"

## Beginner-Friendly Communication Rules

- Start with the visible problem before naming deeper causes.
- Replace dense jargon with short plain-language explanations when possible.
- Explain the fix as cause and result: what was wrong, what changed, and why that helps.
- Avoid listing many speculative theories after the cause is confirmed.
- Mention the validation in a reassuring, concrete way.
